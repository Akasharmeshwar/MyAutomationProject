# MyAutomationProject

This repository contains multiple **QA Automation Projects** built with **Java, Selenium, Playwright, TestNG, and Maven**.  
It is designed for scalable testing in **banking and enterprise domains**, with CI/CD integration and modular architecture.

---
Projects Included
- Rediffmail Automation**  
  Automates login, email workflows, and validations for Rediffmail web application.

- BSE Automation  
  Automates workflows for Bombay Stock Exchange (BSE) portal, including data validation and reporting.
